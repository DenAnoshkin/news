# Новостной сайт

News — это простой сайт/приложение для отображения новостей.
Это не масштабный медийный портал, а базовый «новостной» проект: показывает, как можно организовать frontend + (возможно) backend для публикации и отображения новостей — с базовой структурой, отображением контента и минимальной логикой.

## Технологии и используемые библиотеки / стек проекта

### Backend

- Node.js — серверная платформа
- Express — серверный фреймворк, обработка запросов и взаимодействие с БД
- MongoDB — база данных для хранения только новостей
- Auth0 — сервис аутентификации и авторизации пользователей

### Frontend

- React — построение пользовательского интерфейса
- React Router — маршрутизация между страницами
- React Redux — управление состоянием приложения
- axios — работа с API и HTTP-запросами
- lodash — вспомогательные функции для работы с данными
- JavaScript — логика приложения
- HTML — разметка
- Tailwind CSS — утилитарная система стилей
